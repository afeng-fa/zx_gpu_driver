# PCI ID 与驱动映射关系

## 兆芯 GPU PCI ID 对照表

| PCI Vendor | PCI Device | 驱动 | GPU 型号 | 支持状态 |
|-----------|-----------|------|---------|---------|
| 0x1D17 | 0x3D00 | arise.ko | Arise (Elite3000) | ✅ arise 驱动支持 |
| 0x1D17 | **0x3A04** | **zx.ko** | **KX-6000 C-960 (CHX002)** | ✅ zx 驱动支持 |
| 0x1D17 | 0x3A03 | zx.ko | CHX001 | ✅ zx 驱动支持 |
| 0x6766 | 0x3D00-0x3D0E | arise.ko | Arise 系列 | ✅ arise 驱动支持 |

## 关键区别

- **3D00 系列** = Arise / Elite3000 家族 → 使用 `arise.ko`
- **3A04/3A03** = CHX001/CHX002 家族 → 使用 `zx.ko`
- 两者是**完全不同的 GPU 架构**，驱动不通用

## 验证命令

    # 查看 GPU PCI ID
    lspci -nn | grep VGA
    
    # 查看驱动支持的 PCI ID
    modinfo zx.ko | grep alias
    modinfo arise.ko | grep alias
